Hello. Here is a list of demos for my GitHub repos as well as some old projects I did:

Better YouTube Subtitles

A browser addon written in TypeScript which adds more customization options to YouTube subtitles.

ECWolf-JS

ECWolf, a Wolfenstein 3D source port, ported to WebAssembly using Emscripten.

The List Picker

A simple decision-making tool, written in JavaScript, for picking elements from a list.

The Obsolete OS Keygen

An old-school key generator for outdated operating systems made in Svelte.

The Stream Detector

A Firefox addon that detects streams and assembles readymade commands for capturing. One of my first projects, now with over 40,000 users.

The Voice Memo Repository

A service consisting of a React frontend and Node backend for quickly and easily recording, storing, removing, and sharing short audio recordings made natively in the browser.

The WebSocket IRC Mimic

An IRC-like experience in the browser implemented using WebSockets.
